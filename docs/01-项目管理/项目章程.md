# 族谱系统项目章程

## 项目基本信息
- **项目名称**: 族谱系统 (Family Tree System)
- **项目代码**: FTS-2025
- **项目经理**: 古月
- **项目开始时间**: 2025年7月27日
- **创建日期**: 2025年7月27日
- **最后更新**: 2025年7月27日
- **项目状态**: 规范制定阶段

## 项目目标
开发一个功能完善的族谱管理系统，支持多用户、多族谱管理，提供可视化族谱展示和智能称呼计算功能。

## 技术架构
### 前端技术栈
- Vue.js 3.4+ (前端框架)
- Element Plus 2.4+ (UI组件库)
- Vite 5.0+ (构建工具)
- Pinia 2.1+ (状态管理)
- Vue Router 4.2+ (路由管理)
- Axios 1.6+ (HTTP客户端)
- G6 (AntV) 4.8+ (图形可视化)
- TypeScript 5.0+ (类型系统)

### 后端技术栈
- Python 3.12 (编程语言)
- Django 4.2 (Web框架)
- Django-Ninja 1.0+ (API框架)
- MySQL 8.0 (主数据库)
- Redis 7.0+ (缓存数据库)
- Celery 5.3+ (异步任务队列)
- Pillow 10.0+ (图像处理)

## 项目范围
### 包含内容
- Web应用开发（支持多端访问）
- 用户权限管理系统
- 族谱可视化展示
- 中国传统称呼计算系统
- 数据导入导出功能
- 完整的开发规范体系
- 自动化测试和部署

### 不包含内容
- 移动端原生应用
- 第三方系统集成
- 商业化运营功能
