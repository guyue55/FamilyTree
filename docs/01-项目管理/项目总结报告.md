## 1. 项目概述

### 1.1 项目背景
族谱系统是一个专业的家族族谱管理平台，旨在帮助用户创建、管理和展示家族族谱信息。随着人们对家族文化传承的重视，传统的纸质族谱已无法满足现代化管理需求，因此开发了这个数字化族谱管理系统。

### 1.2 项目范围
- **功能范围**: 用户管理、族谱创建、成员管理、关系建立、图形展示、亲属计算、文件管理
- **技术范围**: 前后端分离架构、Web应用、移动端适配、数据可视化
- **用户范围**: 个人用户、家族管理员、族谱爱好者

### 1.3 项目周期

## 8. 技术难点与解决方案

### 8.1 图形可视化挑战

#### 8.1.1 问题描述
族谱图形展示是系统的核心功能，面临以下挑战：
- 大规模族谱的性能问题
- 复杂关系的布局算法
- 多种展示模式的切换
- 移动端的适配问题

#### 8.1.2 解决方案
```javascript
// 1. 虚拟化渲染解决大规模数据问题
const graph = new G6.Graph({
  container: 'container',
  width: 800,
  height: 600,
  modes: {
    default: ['drag-canvas', 'zoom-canvas', 'drag-node']
  },
  // 启用虚拟化
  enabledStack: true,
  maxZoom: 5,
  minZoom: 0.1
});

// 2. 自定义布局算法
const layout = new G6.Layout.TreeLayout({
  type: 'compactBox',
  direction: 'TB',
  getId: (d) => d.id,
  getHeight: () => 60,
  getWidth: () => 120,
  getVGap: () => 20,
  getHGap: () => 30
});

// 3. 响应式设计
const updateGraphSize = () => {
  const container = document.getElementById('container');
  const width = container.offsetWidth;
  const height = container.offsetHeight;
  graph.changeSize(width, height);
  graph.fitView();
};
```

#### 8.1.3 效果评估
- **性能提升**: 支持1000+节点流畅渲染

亲属关系计算是族谱系统的核心算法，需要解决：

#### 8.2.2 解决方案

```python
class KinshipCalculator:
    """亲属关系计算器"""
    
    def __init__(self):
        self.relationship_rules = self._load_relationship_rules()
        self.dialect_titles = self._load_dialect_titles()
    
    def calculate_relationship(self, person1_id, person2_id):
        """计算两人之间的亲属关系"""
        # 1. 使用BFS查找最短关系路径
        path = self._find_shortest_path(person1_id, person2_id)
        if not path:
            return None
            
        # 2. 根据路径推导关系
        relationship = self._derive_relationship(path)
        
        # 3. 获取对应称谓
        title = self._get_title(relationship)
        
        return {
            'relationship': relationship,
            'title': title,
            'path': path
        }
    
    def _find_shortest_path(self, start_id, end_id):
        """使用BFS查找最短路径"""
        from collections import deque
        
        queue = deque([(start_id, [start_id])])
        visited = {start_id}
        
        while queue:
            current_id, path = queue.popleft()
            
            if current_id == end_id:
                return path
                
            # 获取相关人员
            relatives = self._get_relatives(current_id)
            
            for relative_id in relatives:
                if relative_id not in visited:
                    visited.add(relative_id)
                    queue.append((relative_id, path + [relative_id]))
        
        return None
```

- **性能**: 平均计算时间<50ms
