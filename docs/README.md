## 🎨 原型文件说明

`prototypes/` 目录包含项目的HTML原型文件：

- **族谱测试页面.html** - 族谱功能测试页面
- **族谱页面UI设计稿.html** - 完整的UI设计稿
- **族谱页面原型.html** - 基础原型
- **族谱页面完整原型.html** - 完整功能原型
- **族谱页面开发流程图.html** - 开发流程可视化

这些HTML文件是族谱页面的交互式原型，可以直接在浏览器中打开查看设计效果。

这些文件可以直接在浏览器中打开查看，用于：
- UI/UX设计验证
- 功能原型演示
- 开发参考和测试
