# 族谱页面组件设计规范

## 1. 组件架构总览

### 1.1 组件层次结构
```
FamilyTreePage.vue (主页面容器)
├── FamilyTreeHeader.vue (页面头部)
├── FamilyTreeToolbar.vue (功能工具栏)
├── FamilyTreeSidebar.vue (侧边栏容器)
│   ├── FamilyInfoCard.vue (族谱信息卡片)
│   ├── MemberFilters.vue (成员筛选器)
│   ├── MemberList.vue (成员列表)
│   └── FamilyStats.vue (统计信息)
├── FamilyTreeGraph.vue (图形展示主容器)
│   ├── GraphCanvas.vue (图形画布)
│   ├── GraphNode.vue (节点组件)
│   ├── GraphEdge.vue (连线组件)
│   └── GraphControls.vue (图形控制器)
├── FamilyTreeStatusBar.vue (状态栏)
└── 弹窗组件集合
    ├── MemberDetailDialog.vue (成员详情弹窗)
    ├── AddMemberDialog.vue (添加成员弹窗)
    ├── EditMemberDialog.vue (编辑成员弹窗)
    ├── RelationshipDrawer.vue (关系查询抽屉)
    └── ContextMenu.vue (右键菜单)
```

### 1.2 数据流设计
```
Store (Pinia) ←→ FamilyTreePage ←→ 子组件
     ↓                ↓
API Services    Local State
```
