## 1. 测试概述

### 1.1 测试目标
- **功能完整性**: 验证所有功能模块按需求正常工作
- **性能指标**: 确保系统满足性能要求
- **安全性**: 验证系统安全机制有效性
- **兼容性**: 确保多平台、多浏览器兼容
- **用户体验**: 验证界面友好性和易用性

### 1.2 测试范围
- **功能测试**: 用户管理、族谱管理、关系管理、可视化展示
- **接口测试**: RESTful API接口测试
- **性能测试**: 负载测试、压力测试、并发测试
- **安全测试**: 权限控制、数据安全、SQL注入防护
- **兼容性测试**: 浏览器兼容、设备兼容
- **用户体验测试**: 界面测试、交互测试

### 1.3 测试策略
- **测试驱动开发**: 先写测试用例，再实现功能
- **自动化测试**: 单元测试、集成测试、端到端测试
- **持续集成**: 代码提交自动触发测试
- **分层测试**: 单元测试 → 集成测试 → 系统测试 → 验收测试

    expect(accessResponse.body.code).toBe(10003);
  });
});

# tests/test_api_integration.py
import pytest
from django.test import TransactionTestCase
from django.urls import reverse
from rest_framework.test import APIClient
from rest_framework import status
from django.contrib.auth import get_user_model

User = get_user_model()

class TestFamilyTreeAPIIntegration(TransactionTestCase):
    """
    族谱API集成测试
    测试完整的API调用流程
    """
    
    def setUp(self):
        """测试前置设置"""
        self.client = APIClient()
        self.user = User.objects.create_user(
            username='testuser',
            email='test@example.com',
            password='password123'
        )
    
    def test_complete_family_tree_workflow(self):
        """测试完整的族谱操作流程"""
        # 1. 用户登录
        login_response = self.client.post(reverse('auth:login'), {
            'username': 'testuser',
            'password': 'password123'
        })
        
        self.assertEqual(login_response.status_code, status.HTTP_200_OK)
        token = login_response.data['data']['token']
        
        # 设置认证头
        self.client.credentials(HTTP_AUTHORIZATION=f'Bearer {token}')
        
        # 2. 创建族谱
        family_tree_data = {
            'name': '测试族谱',
            'description': '这是一个测试族谱',
            'family_name': '测试',
            'visibility': 1
        }
        
        create_response = self.client.post(
            reverse('familytree:families-list'),
            family_tree_data
        )
        
        self.assertEqual(create_response.status_code, status.HTTP_201_CREATED)
        family_tree_id = create_response.data['data']['id']
        
        # 3. 添加成员
        member_data = {
            'name': '测试成员',
            'gender': 1,
            'generation': 1,
            'birth_date': '1990-01-01',
            'is_alive': True
        }
        
        member_response = self.client.post(
            reverse('familytree:family-members-list', kwargs={'tree_id': family_tree_id}),
            member_data
        )
        
        self.assertEqual(member_response.status_code, status.HTTP_201_CREATED)
        member_id = member_response.data['data']['id']
        
        # 4. 获取族谱图形数据
        graph_response = self.client.get(
            reverse('familytree:family-tree-graph', kwargs={'tree_id': family_tree_id})
        )
        
        self.assertEqual(graph_response.status_code, status.HTTP_200_OK)
        self.assertIn('nodes', graph_response.data['data'])
        self.assertIn('edges', graph_response.data['data'])
        
        # 5. 删除成员
        delete_response = self.client.delete(
            reverse('familytree:family-member-detail', kwargs={
                'tree_id': family_tree_id,
                'member_id': member_id
            })
        )
        
        self.assertEqual(delete_response.status_code, status.HTTP_204_NO_CONTENT)
