## 1. 产品概述

### 1.1 产品定位
族谱系统是传统纸质族谱的现代化电子版本，旨在将传统家族族谱数字化，以树形结构为核心展示方式，帮助用户创建、管理和展示家族历史。作为新时代的"族谱"，系统不仅保持了传统族谱的核心价值，更通过数字化技术提供了便捷的管理、分享和查询功能。

### 1.2 产品愿景
- 传承传统族谱文化，推动家族历史数字化
- 让每个家族都能轻松创建和维护自己的数字族谱
- 通过现代技术手段保护和弘扬家族文化传统
- 建立家族成员间的数字化连接和互动平台

### 1.3 核心特色
- **树形结构展示**: 以传统族谱的树形结构为基础，提供直观的可视化展示
- **用户创建分享**: 每个用户都可以创建属于自己家族的族谱，并可分享给他人查看
- **真实用户关联**: 族谱中的每个节点/族人都可以关联到系统中的真实注册用户
- **智能称呼查询**: 支持选择任意节点查看与其他族人的称呼关系

### 1.4 目标用户
- **主要用户**: 对家族历史感兴趣的成年人（25-65岁）
- **次要用户**: 家族长辈、年轻一代
- **用户特征**: 具备基本的互联网使用能力，重视家族传统和血缘关系
- **用户规模**: 预计支持1000个家族，每个家族100-1000人

## 2. 功能需求

### 2.1 用户管理
#### 2.1.1 用户注册登录

- [ ] 手机号注册  
- [ ] 密码登录

- [ ] 编辑者：编辑族谱信息
- [ ] 查看者：只读权限

### 2.2 族谱管理

#### 2.2.1 族谱创建
- **功能描述**: 每个用户都可以创建属于自己家族的族谱
- **创建流程**:
  - 填写族谱基本信息（族谱名称、家族姓氏、祖籍地等）
  - 设置族谱可见性（公开、家族内、私有）
  - 添加族谱封面图片和描述
  - 创建始祖节点作为族谱根节点

#### 2.2.2 族谱分享
- **功能描述**: 支持将族谱分享给他人查看
- **分享方式**:
  - **公开分享**: 任何人都可以查看族谱
  - **家族内分享**: 仅家族成员可以查看
  - **私有族谱**: 仅创建者和授权用户可以查看
  - **链接分享**: 生成分享链接，通过链接访问
  - **权限控制**: 设置不同用户的查看和编辑权限

#### 2.2.3 成员管理
- **功能描述**: 管理族谱中的族人信息
- **核心功能**:
  - 添加族人：录入族人基本信息
  - 编辑族人：修改族人详细信息
  - 删除族人：移除族人节点（软删除）
  - 关联用户：将族人节点关联到系统真实用户
  - 批量导入：支持Excel等格式批量导入族人信息

#### 2.2.4 关系建立
- **功能描述**: 建立和维护族人之间的血缘关系
- **关系类型**:
  - 血缘关系：父子、母子、兄弟姐妹等
  - 婚姻关系：夫妻关系
  - 收养关系：养父母子女关系
- **操作功能**:
  - 拖拽建立关系
  - 关系验证和冲突检测
  - 关系历史记录和变更追踪

#### 2.2.5 协作编辑
- **功能描述**: 支持多用户协作编辑族谱
- **权限级别**:
  - 创建者：完全控制权限
  - 管理员：编辑和管理权限
  - 编辑者：添加和修改族人信息
  - 查看者：仅查看权限
- **协作功能**:
  - 实时编辑冲突检测
  - 变更历史记录
  - 编辑通知和审核机制

### 2.3 族谱展示

#### 2.3.1 树形结构展示
- **功能描述**: 以传统族谱的树形结构为基础，提供直观的可视化展示
- **核心特性**:
  - 支持多种族谱布局方式（垂直树、水平树、辐射树等）
  - 节点展示族人基本信息（姓名、照片、生卒年等）
  - 连线表示血缘关系和婚姻关系
  - 支持缩放、拖拽、搜索等交互操作
  - 响应式设计，适配不同屏幕尺寸

#### 2.3.2 节点详情查看
- **功能描述**: 点击族谱节点查看族人详细信息
- **包含信息**:
  - 基本信息：姓名、性别、生卒日期、照片
  - 详细信息：出生地、居住地、职业、学历
  - 个人简介和主要成就
  - 关联的系统用户信息（如已关联）

#### 2.3.3 称呼关系查询
- **功能描述**: 选择任意节点/族人，查看与其他节点/族人的称呼关系
- **核心特性**:
  - 点击选择基准族人
  - 高亮显示与其他族人的关系路径
  - 显示标准称呼和方言称呼
  - 支持双向称呼查询（A对B的称呼 / B对A的称呼）
  - 提供称呼解释和使用说明

#### 2.3.4 族谱导航
- **功能描述**: 提供便捷的族谱浏览和导航功能
- **包含功能**:
  - 世代导航：按世代浏览族人
  - 快速搜索：按姓名、关键词搜索族人
  - 族谱概览：显示族谱统计信息
  - 书签功能：收藏重要族人节点

### 3.3 可用性需求
